## Висновок:

Для виконання практичного завдання 3 було взято 2 статті і за допомогою використання timeit було здійснено пошук 2 підрядків у статтях, а саме:
- Існуючого в обох статтях ("Література")
- Неіснуючого в обох статтях ("Літературна")

Пощук здійснювався за допомогою наступних алгоритмів:
- Кнута-Морріса-Пратта
- Рабіна-Карпа
- Боєра-Мура

Для більш об'єктивного результату, замір загального часу проводився шляхом виконання кожного алгоритму різну кількість разів а саме:
- 1 
- 100
- 1000

### Отримавши результати, можу зазначити наступне:
Алгоритм Бойера–Мура є найшвидшим з вищеперерахованих, а його ефективність зумовлена тим, що він пропускає значні частини тексту завдяки евристикам «поганого символу» та «гарного суфікса». Також наглядно продемонстровано швидкість виконання багаторазового пошуку у порівнянні з іншими алгоритмами як для існуючих, так і для неіснуючих рядків.
